# Weather App
A simple and responsive weather application that provides real-time weather information and forecasts for any location

# Description
This is a weather application built to display current weather condition , temperature , and humidity.Users can search for any city and get instant weather updates powered by the OpenWeather API.

# Technologies used
- **Frontend: ** HTML , CSS , JavaScript
- **API: **OpenWeather API
- **Tools: **VS Code , Git , GitHub
